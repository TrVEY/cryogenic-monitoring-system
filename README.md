# Cryogenic Monitoring System

This project is a Python-based real-time monitoring system developed for experimental cryogenic testing environments.

## Features
- Thermocouple data acquisition
- Real-time temperature visualization
- Automated data logging
- CSV export
- Experimental data analysis

## Technologies
- Python
- Arduino
- MATLAB
- Sensors / Thermocouples

## Purpose
The system was developed to support experimental testing and monitoring in a cryogenic gas separation setup.
